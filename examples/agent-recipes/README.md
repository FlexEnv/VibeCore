# Agent recipes

These recipes are uncompiled reference material for coding agents. They are not
features of the current application and must never be mounted, linked from the
application, or copied wholesale.

Read only the recipes that match the requested product:

- `persisted-domain-slice.md`: validated EF Core model, API, generated client,
  and complete React query states.
- `role-aware-workflow.md`: backend-enforced actions with matching UI affordances.
- `connected-data-dashboard.md`: read-only PostgreSQL or ClickHouse access
  through a Flex-provided server-side connection.
- `scheduled-work.md`: typed, idempotent Quartz work with retries and history.

Replace every illustrative name with the real domain language from the planning
conversation. Preserve VibeCore's SSO, antiforgery, generated-client, and
scheduling foundations.
